## co-animal

- 現地では聞けない生き物の裏話ができる
- 生き物の日常を飼育員の声で届ける音声配信サービス

## co-animalの提供機能

- 音声を聞く機能：飼育員の発信を聞くことができる
- いいね機能：飼育員の発信に「いいね」をつけることができる
- コメント機能：飼育員の発信に「コメント」をつけることができる
